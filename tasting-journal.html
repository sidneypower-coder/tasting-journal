貼入你的 Apps Script Web App URL，儲存和讀取都會直接走 Google Sheets。
更新 Apps Script（必要步驟）
需要換成新版 Script，才能支援讀取紀錄功能。URL 不用換，只需更新程式碼。
1
點下方「複製新版 Script」
2
Google Sheet → 擴充功能 → Apps Script → 全選舊程式碼刪掉 → 貼上 → 儲存
3
部署 → 管理部署作業 → 點鉛筆圖示編輯 → 版本選「新版本」→ 部署
4
URL 不變，回來點「測試連線」確認
✓ 已複製！去 Apps Script 貼上並重新部署
